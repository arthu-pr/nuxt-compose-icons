# Contributing to nuxt-compose-icons

Thank you for your interest in contributing!

## Getting started

1. Fork the repository and clone it locally.
2. Install dependencies: `pnpm install`
3. Start the dev environment: `pnpm dev`

## Development workflow

- Source code lives in `packages/nuxt/`
- Docs are in `docs/`
- Examples are in `examples/`

Run tests before submitting:

```bash
pnpm test
```

## Submitting changes

1. Create a branch from `dev`: `git checkout -b feature/my-feature`
2. Make your changes and add tests where relevant.
<!-- 3. Commit using [Conventional Commits](https://www.conventionalcommits.org/): `feat:`, `fix:`, `docs:`, etc. -->
3. Open a pull request targeting the `dev` branch.

## Reporting bugs

Open an issue and include:

- nuxt-compose-icons version
- Nuxt version
- Steps to reproduce
- Expected vs actual behavior

## Questions

Open a [discussion](https://github.com/arthu-pr/nuxt-compose-icons/discussions) for questions or ideas.
